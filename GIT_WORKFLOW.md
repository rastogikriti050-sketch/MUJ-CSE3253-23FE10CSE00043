# Git Workflow

## GitFlow vs GitHub Flow
GitFlow uses multiple branches (main, develop, feature, release, hotfix).
GitHub Flow uses only main and feature branches.

## Rebase vs Merge
Rebase gives clean linear history.
Merge preserves full history.

## Commit Message Convention
feat: new feature
fix: bug fix
docs: documentation
